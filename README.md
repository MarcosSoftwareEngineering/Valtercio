# 📊 Relatório Estratégico: Integração Claude & Excel

Um relatório dinâmico e interativo desenvolvido para apresentar soluções de Inteligência Artificial integradas ao fluxo de trabalho no Microsoft Excel.

[![Acessar Relatório](https://img.shields.io/badge/Acessar-Relatório_Online-f97316?style=for-the-badge&logo=github)](https://marcossoftwareengineering.github.io/Valtercio/)

## 🎯 Sobre o Projeto

Este projeto foi criado como um artefato de consultoria tecnológica para um cliente cujos requisitos e operações diárias dependem fortemente de planilhas eletrônicas. O objetivo é demonstrar, de forma clara e visualmente engajadora, as vantagens práticas, os primeiros passos e o retorno sobre o investimento (ROI) ao adotar o **Claude for Excel** (Anthropic).

A interface foi projetada para substituir o formato estático de um PDF tradicional, oferecendo uma experiência web interativa que reforça o valor agregado e o profissionalismo da proposta técnica.

## 🚀 Tecnologias Utilizadas

*   **HTML5 & CSS3:** Estruturação semântica e animações nativas (`@keyframes`).
*   **Tailwind CSS:** Framework utilitário (via CDN) para design moderno, limpo e totalmente responsivo.
*   **Canvas Confetti:** Biblioteca JavaScript para efeitos visuais imersivos na tela de abertura.
*   **Open Graph (OG) & Twitter Cards:** Meta tags configuradas no `<head>` para garantir *rich previews* (título, descrição e miniatura) ao compartilhar o link via WhatsApp, LinkedIn e outras plataformas.

## ✨ Funcionalidades e Destaques

*   **Splash Screen Animada:** Tela de carregamento inicial com efeito de explosão de confetes e transição em *fade-out* para o conteúdo principal.
*   **Design Responsivo:** Layout fluido que garante excelente leitura tanto em dispositivos móveis quanto em desktops.
*   **Hierarquia de Informação:** Casos de uso práticos ("os 5 primeiros pedidos que valem o investimento") e comparativo estruturado de planos e preços.
*   **Otimização de Compartilhamento:** *Thumbnails* dinâmicas que profissionalizam o primeiro contato do cliente com o link.
